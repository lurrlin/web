HTML

Html é uma linguagem de marcação utiliada na web que usa tags para "definir" como o texto é mostrado em tela

Um documento html é sempre composto por 6 tags padronizadas <!DOCTYPE>, <html>, <head>, <meta charset>, <title>, <body>.

    Definição das Tags
  • <!DOCTYPE> - Define o tipo do arquivo, no caso "html".

  • <head> - É o cabeçalho do documento, ali vão as especificações do documento, e sua aba <style> (se houver).

  • <meta charset> - É a fonte dos caracteres do documento (UTF-8) que é como o sistema interpreta os bits para formar caracteres.

  • <title>- É o nome do documento/página.

  • <body> - É o corpo do documento, aqui vai todo o conteúdo da página (imagens, textos, links, etc.).
