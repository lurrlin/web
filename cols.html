1. Estrutura básica do HTML

Documento HTML começa com <!doctype html> e contém as seções <head> e <body>.

Elementos comuns

  • <h1>…</h1> a <h6> — títulos hierárquicos.
  • <p> — parágrafos.
  • <a href=""> — link; atributos: href, target, rel.
  • <img src="" alt=""> — imagem; atributos: src, alt, width, height, loading.
  • <ul>/<ol>/<li> — listas não ordenadas/ordenadas.
  • <table> — tabelas com <thead>, <tbody>, <tr>, <th>, <td>.
  • <form> — formulários com <input>, <select>, <textarea>, <button>.

Semântica HTML

Use tags semânticas: <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>. Isso melhora acessibilidade e SEO.

3. CSS — propriedades e exemplos

Abaixo há exemplos e comentários sobre as propriedades CSS mais usadas.

Seletores (exemplos)

  • Elemento: p { ... }
  • Classe: .classe { ... }
  • ID: #id { ... }
  • Descendente: .pai .filho { ... }
  • Filho direto: .pai > .filho { ... }
  • Atributo: a[target="_blank"] { ... }
  • Pseudo-classes: a:hover, input:focus
  • Pseudo-elementos: p::before, p::after

Layout

Flexbox: display:flex, flex-direction, justify-content, align-items, gap.

Grid: display:grid, grid-template-columns, grid-template-rows, gap.

Box model

Propriedades: width, height, padding, border, margin, box-sizing. Lembre-se que padding e border afetam o tamanho se box-sizing for content-box.

Tipografia

Propriedades: font-family, font-size, font-weight, line-height, letter-spacing, text-align, text-decoration, text-transform.

Efeitos visuais

Propriedades: background, border-radius, box-shadow, opacity, filter, transform, transition, animation.

Outros utilitários

  • overflow, white-space, text-overflow
  • position: static, relative, absolute, fixed, sticky
  • z-index para empilhamento
  • cursor, pointer-events

Exemplo rápido (classe)

/* botão de destaque */
.btn-destaque {
  background: var(--accent);
  color: white;
  padding: .6rem 1rem;
  border-radius: 6px;
  border: none;
  cursor: pointer;
}
