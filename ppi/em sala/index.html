Um pouco de texto. Continuação do texto e mais texto de novo
  1. git init - cria um repositório git
  2. git add - adiciona as modificações feitas para commit
  3. git log - mostra todos os commits feitos na branch atual. "--oneline --all"(identifica os commits com a primeira linha de cada, menos verboso)
  4. git checkout "branch" - Troca da branch atual para a especificada
  5. git branch --all - Lista todas as branches remotas criadas no repositório
  6. git gc - Retira todos os commits onde não há apontamentos para eles
  7. git reset "branch" - desvincula uma branch em específico e posiciona todos os seus commits para "esperar deleção"
  8. git "cmd" TAB - Lista os comandos disponíveis para este cmd
  9. git remote - mostra os repositórios remotos disponíveis(caso tenha algum)
